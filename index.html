Псевдоклассы

Псевдоклассы определяют состояние элемента выбранного селектором. Псевдокласс ставится после класса вместе с двоеточием.

Синтаксис: селектор:псевдо_класс { свойство: значение; }

Более новый синтаксис не всегда поддерживается браузерами (два двоеточия):

Синтаксис: селектор::псевдо_класс { свойство: значение; }

Ссылки

Для ссылок существует несколько псевдоклассов:

  • link — определяет непосещённые ссылки
  • visited — определяет уже посещённые ссылки

Динамические псевдоклассы

Динамические псевдоклассы применяются когда с элементом происходит какое-то событие. Они так же используются для ссылок, но применимы к другим элементам и более часто необходимы для элементов форм.

  • active — когда пользователь нажимает на элемент (например нажимает на ссылку).
  • hover — когда курсор наводится на элемент (например пользователь наводит мышь на ссылку).
  • focus — когда элемент находится в фокусе, он выбран либо готов к вводу в него данных (в случае текстовых полей). Можно менять активный элемент на странице используя клавишу Tab. Если это элемент формы или ссылка можно поставить на этот элемент фокус просто кликнув на него.

See the Pen Dynamic pseudo classes by Aleksey Korovin (@alekskorovin) on CodePen.

Псевдоклассы для выбора элемента

Существую псевдоклассы позволяющие выбрать элементы по порядку либо порядковому номеру: например первый, последний, второй, третий, чётный/нечётный элемент и другие (CSS3 псевдоклассы).

  • first-child — выбирает первый элемент
  • last-child — выбирает последний элемент
  • nth-child(2) — выбирает второй элемент
  • nth-child(2n) — выбирает каждый второй элемент
  • nth-child(even) — выбирает чётный элемент
  • nth-child(odd) — выбирает нечётный элемент

See the Pen Child pseudo classes by Aleksey Korovin (@alekskorovin) on CodePen.

Псевдоэлементы before и after

Псевдоэлементы, которые используется для добавления контента до (before) и после (after) содержимого элемента, к которому он добавляется.

Эти псевдоэлементы и их контент используются для декоративных целей, например можно выводить все ссылки в явном виде при печати страницы.

Псевдоэлементы before и after это очень мощный инструмент для добавления декоративных CSS-свойств.
