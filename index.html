Псевдоклассы

Псевдоклассы определяют состояние элемента выбранного селектором. Псевдокласс ставится после класса вместе с двоеточием.

Синтаксис: селектор:псевдо_класс { свойство: значение; }

Более новый синтаксис CSS3, не всегда поддерживается браузерами (два двоеточия).

Синтаксис: селектор::псевдо_класс { свойство: значение; }

  • link — определяет непосещённые ссылки
  • visited — определяет уже посещённые ссылки

Динамические псевдоклассы

Динамические псевдоклассы применяются когда с элементом происходит какое-то событие. Они так же используются для ссылок, но применимы к другим элементам и более часто необходимы для элементов форм.

  • active — когда пользователь нажимает на элемент (например нажимает на ссылку).
  • hover — когда курсор наводится на элемент (например пользователь наводит мышь на ссылку).
  • focus — когда элемент находится в фокусе, он выбран либо готов к вводу в него данных (в случае текстовых полей). Можно менять активный элемент на странице используя клавишу Tab. Если это элемент формы или ссылка можно поставить на этот элемент фокус просто кликнув на него.

See the Pen Dynamic pseudo classes by Aleksey Korovin (@alekskorovin) on CodePen.

Псевдоклассы для выбора элемента

Существую псевдоклассы позволяющие выбрать элементы по порядку либо порядковому номеру: например первый, последний, второй, третий, чётный/нечётный элемент и другие (CSS3 псевдоклассы).

  • first-child — выбирает первый элемент
  • last-child — выбирает последний элемент
  • nth-child(2) — выбирает второй элемент
  • nth-child(2n) — выбирает каждый второй элемент
  • nth-child(even) — выбирает чётный элемент
  • nth-child(odd) — выбирает нечётный элемент

Псевдоэлементы before и after

Псевдоэлементы, которые используется для добавления контента до (before) и после (after) содержимого элемента, к которому он добавляется.

Эти псевдоэлементы и их контент используются для декоративных целей, например можно выводить все ссылки в явном виде при печати страницы.

Псевдоэлементы before и after это очень мощный инструмент для добавления декоративных CSS-свойств. Кроме того можно делать всплывающие подсказки, размещая текст для подсказок в атрибутах элемента.

Псевдоэлементы before и after испльзуются только вместе со свйойством content в котором размещается дополнительный контент, выводимый из CSS в HTML либо использующий значения атрибутов элемента.

Значения свойства content

  • 'строка' — возможно размещение любого текста в кавычках.
  • url(путь_к_изображению) — возможно указывать URL (путь, адрес) к изображению в кавычках.
  • open-quote — открывающая кавычка.
  • close-quote — закрывающая кавычка.
  • attr(название_атрибута) — выводит текст указанный как свойсто атрибута.

Флоаты

Для того чтобы сделать блок "плавающим" (float - floating) для элемента необходимо применить свойство float, которое может принимать три значения:

  • left — блок прибивается к левому краю доступной области и контент обтекает блок справа.
  • right — блок прибивается к правому краю доступной области и контент обтекает блок слева.
  • none — отменяются свойства left/right.

Позиционирование

Чтобы задать позиционирование элемента используется свойсво position. По умолчанию у элементов значение position это static.

Все значения свойства position:

  • static — нормальное позиционирование, элемент участвует в общем потоке элементов (элементы идут один под другим).
  • relative — позиционирование похожее на static, но при этом свойстве элемент может быть вынесен на более высокий или более низкий слой с помощью свойства z-index, а так же элемент может быть смещён относительно своего нормального положения с помощью свойств:
    • top — отступ сверху.
    • left — отступ слева.
    • right — отступ справа.
    • bottom — отступ снизу.
    При position: relative элементом резервируется то же место, что и до смещения элемента.
    В этом случае свойства top, left, right, bottom работают так же как и внешний отступ (margin).
  • absolute — при этом элемент вообще выпадает из потока других элементов, и его позиционирования можно менять отступами margin и свойствми top, left, right, bottom. Этот элемент перестает влиять на другие элементы и не подвержен влияию других элементов кроме родителя с position: relative; При этом позиционировании свойства ведут себя так:

    • Если элемент не находится внутри другого элемента с position: relative и ни одно из свойств top, left, right, bottom не задано, то элемент повисает в "воздухе" над остальными элементами идущими в общем потоке.

    • Если элемент не внутри другого элемента с position: relative и задано одно из свойств top, left, right, bottom, то эти свойства работают так:

      • top: 10px; — элемент прилипает к верхней части экрана body и смещается вниз на 10px.
      • left: 1em; — элемент прилипает к левой части экрана и смещается впрвао на 1em.
      • right: 0; — элемент прилипает к правой части экрана.
      • bottom: 1rem; — элемент прилипает к нижней части видимой области (НЕ ЭКРАНА) и смещается вверх на 1rem.

    • Если элемент внутри другого элемента с position: relative то "прилипание" происходит не к экрану, а к краям этого родительского элемента (смещается элемент так же).

  • fixed — при таком позиционировании элемент ведёт себя почти так же как и при absolute, но "прилипает" к границам экрана вне зависимости от родителей с position: relative;

Свойства top, left, right, bottom

Значения свойств задаются в размерных CSS-единицах (px, em, rem, % ...), если задаётся значение в процентах, используются размеры родителя относительно которого эти проценты рассчитываются.

Для сброса ранее заданных значений использвется свойство auto: чтобы перебить left: 50%; надо задать left: auto;.

Свойство z-index

Свойство z-index задаёт порядок слоя в котором находится элемент при position relative, absolute, fixed. Значением может быть целое число либо значение по умолчанию auto.

Обзор новых семантических тегов

Структура со смыслом

Заголовки и подзаголовки прекрасно работают для простых страниц и для разделов. Чтобы разбить содержимое страницы на разделы и более понятные семантические (несущие смысл) структуры есть ряд специальных элементов.

Блочные элементы div позволяют структурировать содержание страницы и сделать отображение элементов с помощью CSS таким каким нам нужно по дизайну, но тег
<div>Содержание блока</div> не несёт никакой смысловой нагрузки.

В HTML5 возможно задать отдельные части страницы такие как:
шапка (хедер, header), подвал (футер, footer), основную часть, разделы (секции) и навигацию.

Статьи и разделы

Элемент article

Элемент article используется для разметки отдельного независимого блока контента относящегося по смыслу к одной теме, и который может быть использован отдельно от остальных частей страницы (например вставлен в RSS-ленту), примеры:

  • Статья блога содержащая свой заголовок, подзаголовки, дату публикации.
  • Несколько элементов article — список коротких заметок (как в газете).
  • Комментарий пользователя, содержащиий имя поьзователя, дату публикации.
  • Виджеты (блок с прогнозом погоды).

Элемент article может включать в себя собственный header, footer.

Особенность Даже если на странице уже есть h1, внутри элемента article мы можем задать собственный h1.

Тени

CSS3 позволяет добавлять тень (shadow) к блокам и тексту с помощью свойств: box-shadow и text-shadow.

Каноническое демо CSS3, please! использует оба свойства.

Свойство box-shadow

Одной строкой задаются все параметры тени блока. Первые четыре параметра задаются в размерных единицах CSS, последний в цветовом формате. Возможно использование нескольких теней, указывая их параметры через запятую, при наложении, первая тень будет размещаться выше, последняя ниже. При скругленных углах элемента, тень так же становится скруглённой.

Синтаксис box-shadow: [inset] сдвиг_X сдвиг_Y радиус_размытия растяжение цвет;
  • сдвиг_X — смещение тени по горизонтали, положительное значение сдвигает тень вправо, со знаком минус — влево. обязателен.
  • сдвиг_Y — смещение тени по вертикали, положительное значение сдвигает тень вниз, со знаком минус — вверх. обязателен.
  • радиус_размытия — большее значение с большей степенью размывает тень, значение по умолчанию устанавливается равным 0 — тень полностью чёткая.
  • растяжение — положительное значение увеличиват тень, отрицательное — уменьшает. Значение по умолчанию 0 — тень того же размера.
  • цвет — задаёт цвет тени, может быть задано непрозрачным цветом, а так же с полупрозрачностью в формате RGBA rgba(0, 0, 0, 0.5);.
  • inset — если это значение задано, тень будет рисоваться внутрь элемента.

See the Pen Box shadow by Aleksey Korovin (@alekskorovin) on CodePen.

Свойство text-shadow

Одной строкой задаются все параметры тени текста. Первые три параметра задаются в размерных единицах CSS, последний в цветовом формате. Возможно использование нескольких теней, указывая их параметры через запятую, при наложении, первая тень будет размещаться выше, последняя ниже.

Синтаксис text-shadow: сдвиг_X сдвиг_Y радиус_размытия цвет;
  • сдвиг_X — смещение тени по горизонтали, положительное значение сдвигает тень вправо, со знаком минус — влево. обязателен.
  • сдвиг_Y — смещение тени по вертикали, положительное значение сдвигает тень вниз, со знаком минус — вверх. обязателен.
  • радиус_размытия — большее значение с большей степенью размывает тень, значение по умолчанию устанавливается равным 0 — тень полностью чёткая.
  • цвет — задаёт цвет тени, может быть задано непрозрачным цветом, а так же с полупрозрачностью в формате RGBA rgba(0, 0, 0, 0.5);.

Градиенты

Градиент это изображение на котором цвета плавно переходят из одного в другой (градиенты называют цветовыми растяжками). Чтобы не использовать фоновое изображение для градиента, ведь для загрузки изображения требуется дополнительный запрос на сервер, в CSS добавили четыре функции, позволяющих нарисовать нужный градиент используя заданные параметры.

Чтобы отобразить градиент используются свойства background или background-image, и вместо изображения использовать одну из четырёх функций: linear-gradient, radial-gradient, repeating-linear-gradient, repeating-radial-gradient.

Типы поддерживаемых в CSS градиентов:

  • линейный градиент (linear-gradient) — переход между цветами идёт в одну сторону. Простейший линейный градиент имеет два цвета (две точки), может быть задано несколько точек.
    повторяющийся линейный градиент (repeating-linear-gradient) — позволяет повторять линейный градиент.
  • радиальный градиент (radial-gradient) — переход между цветами идёт как увеличивающаяся окружность.
    повторяющийся радиальный градиент (repeating-radial-gradient) — позволяет повторять радиальный градиент.

Линейные градиенты

По умолчанию направление градиента сверху вниз.

Синтаксис: background: linear-gradient(цвет_1, цвет_2);

Но можно задавать направление градиента добавив первым параметром строку to направление (top, left, right, bottom):

Синтаксис: background: linear-gradient(to направление, цвет_1, цвет_2);

See the Pen Gradient direction by Aleksey Korovin (@alekskorovin) on CodePen.

Можно задавать угол как направление, тогда добавляемя строка будет состоять из двух слов.

Синтаксис: background: linear-gradient(to верх_низ лево_право, цвет_1, цвет_2);

See the Pen Gradient direction corner by Aleksey Korovin (@alekskorovin) on CodePen.

Можно вместо направления для угла задавать градус.

Синтаксис: background: linear-gradient(градус_deg, цвет_1, цвет_2);

See the Pen Gradient direction corner angle by Aleksey Korovin (@alekskorovin) on CodePen.
